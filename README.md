# 💅 Little Nails Luxury - Sistema Web de Gestión

Proyecto académico desarrollado siguiendo el marco de trabajo **SCRUM**.  
El sistema permite la gestión de diseños de uñas personalizadas, usuarios y pedidos.

## 📌 Objetivo del Proyecto
Desarrollar un sistema web que permita:
- Gestionar clientes y diseñadoras.
- Mostrar catálogo de diseños de uñas.
- Registrar pedidos personalizados.
- Control de usuarios y roles con login seguro.
