# Backlog - Historias de Usuario

- **HU1**: Como cliente quiero ver un catálogo con diseños de uñas personalizadas, para elegir el estilo que más me gusta antes de hacer mi pedido.
- **HU2**: Como cliente quiero iniciar sesión de manera segura, para proteger mis datos personales.
- **HU3**: Como diseñadora quiero dar de alta, editar o eliminar diseños, para mantener la información actualizada.
- **HU4**: Como administradora quiero gestionar usuarios y roles, para tener control sobre el sistema.
